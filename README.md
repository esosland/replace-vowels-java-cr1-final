#_Replace Vowels Java App_#
###By Elissa Sosland###

##**Description**##
This is a Java app that lets a user enter a phrase and then returns the phrase with dashes in place of vowels.

##**Setup**##
* Clone the repository
* Gradle run the app
* Open localhost:4567 in web browser

##**Technologies Used**##
* Java
* Spark
* Velocity
* FluentLenium
* CSS
* BootStrap
* HTML
